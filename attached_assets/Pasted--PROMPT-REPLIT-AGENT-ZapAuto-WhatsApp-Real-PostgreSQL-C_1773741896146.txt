# PROMPT REPLIT AGENT — ZapAuto WhatsApp Real + PostgreSQL

## CONTEXTO
Este projeto é o ZapAuto rodando no Replit. O problema atual é que
o WhatsApp está em modo demo porque o IP do Replit é bloqueado pelo
WhatsApp quando usamos whatsapp-web.js (Puppeteer).

A solução é trocar para a biblioteca **Baileys** que usa WebSocket
direto e funciona em qualquer ambiente incluindo Replit.

O Replit já tem PostgreSQL disponível via variável DATABASE_URL.

### Reescrever `src/lib/whatsapp.ts` completo com Baileys:

```typescript
import 'dotenv/config'
import makeWASocket, {
  DisconnectReason,
  useMultiFileAuthState,
  fetchLatestBaileysVersion,
  makeCacheableSignalKeyStore,
  proto,
  WASocket,
} from '@whiskeysockets/baileys'
import { Boom } from '@hapi/boom'
import qrcode from 'qrcode'
import { getWhitelistByPhone, saveQueryHistory } from './db.js'
import { interpretarTextoLivre } from './query.js'
import pino from 'pino'

const logger = pino({ level: 'silent' })

// ── Estado singleton ──────────────────────────────────────
let sock: WASocket | null = null
let lastQrRaw: string | null = null
let connected = false
let statusText = 'disconnected'
let currentNumber: string | null = null
let reconnectAttempts = 0
const MAX_RECONNECT = 5

// ── Helpers ───────────────────────────────────────────────
function inferTipo(texto: string): 'material' | 'op' | 'urgente' {
  const t = texto.trim()
  if (t.startsWith('!')) return 'urgente'
  if (/^op\s*\d+/i.test(t) || /^\d{6,}$/.test(t.replace(/\s/g, ''))) return 'op'
  return 'material'
}

// ── Iniciar cliente ───────────────────────────────────────
export async function startWhatsappClient(): Promise<void> {
  if (sock && connected) {
    console.log('[WhatsApp] Já conectado')
    return
  }

  const { state, saveCreds } = await useMultiFileAuthState('.baileys_auth')
  const { version } = await fetchLatestBaileysVersion()

  sock = makeWASocket({
    version,
    auth: {
      creds: state.creds,
      keys: makeCacheableSignalKeyStore(state.keys, logger),
    },
    logger,
    printQRInTerminal: false,
    browser: ['ZapAuto', 'Chrome', '120.0.0'],
    connectTimeoutMs: 60_000,
    defaultQueryTimeoutMs: 60_000,
    keepAliveIntervalMs: 25_000,
    markOnlineOnConnect: false,
  })

  // QR Code
  sock.ev.on('connection.update', async (update) => {
    const { connection, lastDisconnect, qr } = update

    if (qr) {
      lastQrRaw = qr
      statusText = 'qr_ready'
      console.log('[WhatsApp] QR gerado — escaneie no painel')
    }

    if (connection === 'open') {
      connected = true
      statusText = 'connected'
      lastQrRaw = null
      reconnectAttempts = 0
      currentNumber = sock?.user?.id?.split(':')[0] ?? null
      console.log('[WhatsApp] ✅ Conectado! Número:', currentNumber)
    }

    if (connection === 'close') {
      connected = false
      currentNumber = null
      const reason = (lastDisconnect?.error as Boom)?.output?.statusCode
      console.log('[WhatsApp] Desconectado. Razão:', reason)

      const shouldReconnect = reason !== DisconnectReason.loggedOut
      if (shouldReconnect && reconnectAttempts < MAX_RECONNECT) {
        reconnectAttempts++
        console.log(`[WhatsApp] Reconectando (tentativa ${reconnectAttempts})...`)
        setTimeout(() => startWhatsappClient(), 5000)
      } else if (reason === DisconnectReason.loggedOut) {
        statusText = 'logged_out'
        sock = null
        console.log('[WhatsApp] Sessão encerrada. Escaneie o QR novamente.')
      }
    }
  })

  // Salvar credenciais
  sock.ev.on('creds.update', saveCreds)

  // Mensagens recebidas
  sock.ev.on('messages.upsert', async ({ messages, type }) => {
    if (type !== 'notify') return

    for (const msg of messages) {
      // Ignorar mensagens próprias e de grupos
      if (msg.key.fromMe) continue
      if (msg.key.remoteJid?.endsWith('@g.us')) continue

      const from = msg.key.remoteJid ?? ''
      const phone = from.replace('@s.whatsapp.net', '').replace('@c.us', '')
      const texto = msg.message?.conversation
        ?? msg.message?.extendedTextMessage?.text
        ?? ''

      if (!texto.trim()) continue

      console.log(`[WhatsApp] Mensagem de ${phone}: "${texto}"`)

      // Verificar whitelist
      const contato = await getWhitelistByPhone(phone)
      if (!contato || !contato.active) {
        console.log(`[WhatsApp] ${phone} não está na whitelist — ignorando`)
        continue
      }

      try {
        const resultado = await interpretarTextoLivre(texto)

        // Responder
        await sock?.sendMessage(from, { text: resultado.resposta })
        console.log(`[WhatsApp] Resposta enviada para ${phone}`)

        // Salvar histórico
        await saveQueryHistory({
          phone,
          type: inferTipo(texto),
          query: texto,
          found: resultado.found,
          response: resultado.resposta,
        })
      } catch (err) {
        console.error('[WhatsApp] Erro ao processar mensagem:', err)
      }
    }
  })
}

// ── Desconectar ───────────────────────────────────────────
export async function disconnectWhatsappClient(): Promise<void> {
  if (!sock) return
  await sock.logout().catch(() => {})
  sock = null
  connected = false
  statusText = 'disconnected'
  currentNumber = null
  lastQrRaw = null
  console.log('[WhatsApp] Desconectado manualmente')
}

// ── Getters ───────────────────────────────────────────────
export function getWhatsappStatus() {
  return { connected, status: statusText, number: currentNumber }
}

export function getWhatsappQrRaw(): string | null {
  return lastQrRaw
}

export async function listWhatsappContacts() {
  if (!sock || !connected) {
    throw Object.assign(new Error('WhatsApp não conectado'), { status: 400 })
  }
  // Baileys não tem getContacts() direto — retorna vazio por ora
  return []
}
```

## TAREFA 2 — Atualizar rota do QR

Em `src/routes/whatsapp.ts`, o endpoint GET /api/whatsapp/qr
deve converter o QR raw para PNG base64 na hora da requisição:

```typescript
import qrcode from 'qrcode'
import {
  getWhatsappStatus,
  getWhatsappQrRaw,
  startWhatsappClient,
  disconnectWhatsappClient,
  listWhatsappContacts,
} from '../lib/whatsapp.js'

router.get('/qr', async (_req, res) => {
  const raw = getWhatsappQrRaw()
  if (!raw) {
    res.status(404).json({ ok: false, error: 'QR não disponível. POST /connect primeiro.' })
    return
  }
  // Converte string bruta do Baileys para PNG base64
  const dataUrl = await qrcode.toDataURL(raw, { width: 300, margin: 2 })
  const base64 = dataUrl.replace('data:image/png;base64,', '')
  res.json({ qr: base64 })
})
```

## TAREFA 3 — Ativar PostgreSQL com Drizzle

O Replit fornece DATABASE_URL automaticamente como variável de
ambiente. Usar ela para persistir tudo.

Em `src/lib/db.ts`, garantir que quando DATABASE_URL existir,
use o Drizzle com pg:

```typescript
import 'dotenv/config'
import { drizzle } from 'drizzle-orm/node-postgres'
import { Pool } from 'pg'
import * as schema from '../schema/index.js'

let db: ReturnType<typeof drizzle> | null = null

export function getDb() {
  if (!db && process.env.DATABASE_URL) {
    const pool = new Pool({ connectionString: process.env.DATABASE_URL })
    db = drizzle(pool, { schema })
    console.log('[DB] PostgreSQL conectado via DATABASE_URL')
  }
  return db
}

export function databaseAvailable(): boolean {
  return !!process.env.DATABASE_URL
}
```

Rodar as migrations ao iniciar o servidor. Em `src/index.ts`,
após criar o app, adicionar:

```typescript
import { migrate } from 'drizzle-orm/node-postgres/migrator'
import { getDb, databaseAvailable } from './lib/db.js'

// Rodar migrations se banco disponível
if (databaseAvailable()) {
  const db = getDb()
  if (db) {
    migrate(db, { migrationsFolder: './drizzle' }).then(() => {
      console.log('[DB] Migrations aplicadas')
    }).catch(err => {
      console.error('[DB] Erro nas migrations:', err)
    })
  }
}
```
